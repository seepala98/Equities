üîç Classification Rules

  • ETFs: Names containing "ETF", "Index Fund"
  • REITs: Names with "REIT", symbols ending ".UN"
  • Bonds: Symbols ending ".DB" (debentures)
  • Warrants: Symbols ending ".WT", names with "warrant"
  • Crypto: Names with "Bitcoin", "Ethereum", "Blockchain"
  • Funds: Names containing "Fund" (after ETF check)
  • Trusts: Names with "Trust", "Income Trust"
  • Preferred: Names with "Preferred", symbols with ".PR"

  4. Default Assignment: Classify as STOCK if no matches
  5. Database Update: Store classification results
